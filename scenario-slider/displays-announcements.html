Scenario Slider

Walk through creating an announcement that appears across your digital displays.

Scenario: There's an unplanned facility closure tomorrow morning. Walk through creating a time-sensitive announcement that shows on all displays starting tonight and expires after tomorrow.

Step 1 of 4
Navigate to Announcements and create a new one.
Select Announcements from the navigation menu and click New Announcement. The configuration panel opens on the right side of the screen where you'll set up the message and display rules.
Step 2 of 4
Enter the message text and set the display mode.
Type your message in the Text field. Then choose the Display Mode: select All Displays to show on every screen, or Tagged to target only playlists and configurations with a specific tag. For a facility-wide alert, choose All Displays.
Step 3 of 4
Add tags and configure the schedule.
Optionally assign tags to organize announcements and control which displays they appear on. Then enable Scheduling — toggle it on and set the start date/time to tonight and the end date/time to after tomorrow. The announcement will automatically disappear when it expires.
Step 4 of 4
Save and confirm the announcement is live.
Click Save Announcement. It appears in the Announcements table with its scheduled window. Use the pencil icon to edit at any time. When the scheduled start time arrives, the announcement begins running across all targeted displays automatically.

✓ Announcement scheduled — it will appear on all displays tonight and auto-expire after tomorrow.
