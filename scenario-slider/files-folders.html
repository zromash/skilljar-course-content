Walk through creating and organizing folders in Hub's Files module.

Scenario: You need to set up a folder structure for your team's documents — a main folder for the season, with a subfolder for game film. Walk through the steps from start to finish.

Step 1 of 5
Navigate to the Files module in Hub.
Open Hub and select the Files module from the navigation. Here you can see all folders and teams you have access to — this is your starting point for all file organization.
Step 2 of 5
Select the team or folder where you want to add the new folder.
Choose the appropriate team or parent folder, then click the Add New Folder button. This scopes the new folder to the right location before you configure it.
Step 3 of 5
Name the folder and set the parent folder (sport).
Give the folder a descriptive name and select the sport as the parent folder. If you're creating a subfolder, select the folder where you want it to live as the parent — this determines where it nests in the hierarchy.
Step 4 of 5
Optionally add notes and pin the folder.
Add any relevant notes to give context about the folder's purpose. Use the pin option if this folder should stay prominent at the top of the list for easy access.
Step 5 of 5
Select Save — your folder appears in the list.
The newly created folder shows up immediately. To edit or delete it later, select the three-dot icon in the top right corner of the folder and choose your action.

✓ Folder created and organized — you set the parent correctly and know how to add subfolders to build out the full structure.
