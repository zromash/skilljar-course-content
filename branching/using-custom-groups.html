Branching Scenario

You've created a custom group. How do you want to use it today?

You're setting up your week in Teamworks Hub and want to use your custom group. Which action are you taking?

You chose

Add a Custom Group to a Calendar Event

1

Navigate to the Calendar module and select Add Event.

2

Enter the event details — title, date, time, and location.

3

In the Invites section, select the three-dot icon.

4

In the pop-up, select the Groups tab and choose your custom group.

5

Select Create Event — all group members are added as attendees automatically.

✓ Your custom group is now invited to the event. No need to add members one by one.


You chose

Send a Message Blast to a Custom Group

1

Navigate to the Messages module and select New Message Blast.

2

Enter your message content in the message field.

3

In the Recipients field, select the three-dot icon.

4

Select the Custom tab and choose your custom group from the list.

5

Select Next to review, then Send to deliver the blast to all group members.

✓ Your message blast is sent to all members of the custom group at once.
