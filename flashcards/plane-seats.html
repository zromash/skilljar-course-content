Click each card to learn the four stages of setting up plane seat assignments.

Stage 1
Create the Airline Vendor
Click to see how →
How it works
Go to the Vendors tab in Travel. Click Add Vendor → enter the airline name, select vendor type, choose the teams using this vendor, and add address and contact info. Save.
Stage 2
Build the Seat Plan
Click to see how →
How it works
Click into the vendor → Seat Plans tab → Add Seat Plan. Name it, enter sections, configure rows and columns. Mark unavailable seats, exit rows, or limited-recline seats. Save the plan.
Stage 3
Assign Seats to the Trip
Click to see how →
How it works
Open the trip → Vendors tab → add the airline vendor. Then go to the Air tab → Add Plane → select your seat plan → enter the flight name. Drag and drop travelers into their seats. Add separate outbound and return legs for round trips.
Stage 4
Download the Manifest
Click to see how →
How it works
Navigate to the Documents tab in the trip. Download the flight manifest and the seating plan as PDF backups — useful for providing physical copies to airlines or staff at the gate.
