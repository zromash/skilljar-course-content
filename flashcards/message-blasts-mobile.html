What happens when a recipient replies to a mobile message blast?

A private 1:1 thread opens between the sender and that recipient — other recipients can't see the reply.

What does a recipient get if they haven't downloaded the Hub mobile app?

They receive a text message — as long as a cell phone number is in their Teamworks Hub profile.

Can you schedule a message blast from the mobile app?

Yes — you can set a future date and time before sending, so messages go out at the right moment.

Can recipients of a message blast see who else received it?

No — message blasts feel like a 1:1 message. Recipients never see the full recipient list.
