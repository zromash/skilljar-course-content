Copy = Exact replica
Copying creates an exact replication of an event — all details, attendees, and settings — and places it on new dates you select. Everything transfers over unchanged.
Duplicate
What does duplicating an event do?
Click to learn →
Duplicate = Template
Duplicating creates a structural template of the event that you can then modify before saving. Use it when you want the same format but need to change details like time, attendees, or location.
Feature 3
Copying multiple events at once

Bulk copy
To copy multiple events in one action, use the three-dot menu on the calendar view. This lets you select a range of events and copy them all to new dates simultaneously.
Feature 4
When to use Copy vs. Duplicate
Click to learn →
Decision guide
Use Copy when repeating the same event exactly (e.g., same practice schedule next week). Use Duplicate when you want a starting point you'll customize (e.g., similar event with different roster).
