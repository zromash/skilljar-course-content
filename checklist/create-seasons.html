Work through each step to create a season in Hub.

Scenario: Your athletic department is starting a new travel season for football. Walk through setting it up in Hub so your team has an organized framework for all upcoming trips.

0 of 6 steps completed

Navigate to Travel → Trips → Manage Seasons
Find the Travel module in the left menu bar, select Trips, then click Manage Seasons to see your existing seasons and access the creation tools.
Review existing seasons
You can rename any existing season by selecting Edit, making your changes, and hitting Save. This is useful when season names need to be updated.
Click "+ Add Season"
The Add Season button opens the season creation form. This is where you'll name the new season and tie it to the correct team.
Select the correct team from the dropdown
Each season is tied to a specific team. Make sure you're selecting the right team — you'll add all trips for that team under this season.
Enter a descriptive name for the season
Use a clear naming convention — for example "Football 2025-26" or "Fall Travel Season." This name will appear across all trips associated with it.
Click Save to create the season
Your new season immediately appears on the Manage Seasons page and is ready for trips to be added to it.

✓ Season created! Your football season is now set up as the organizational framework for all upcoming trips. Next step: add your first trip.
